<H1>
  Todo App
</H1>
<p>Todo App using NodeJs and MongoDb.</p>

## Setup

<h4><b>Clone Repo</b</h4>
  
```bash
$ https://github.com/Enm4nuel/ptb-L1.git
```

## Installation

```bash
$ npm install
```

## Run app

```bash
$ npm run start
```

## Importat look at this
<p>This app build was using</p>
<ul>
  <li> node >= 14.15.0 </li>
  <li> npm >= 6.11.0 </li>
  <li> in case you use yarn >= 1.13.0" </li>
</ul>
